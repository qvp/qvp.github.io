WikiItem.com (срок 1 год и 2 месяца)

Торговая площадка, в которой любой желающий может создать собственный магазин. База насчитывает более 550 000 товаров, которые можно продавать в своем магазине. Свои товары можно загрузить через REST API или добавить вручную, через конструктор. Конструктор помогает быстро добавлять товары, он автоматически ищет и предлагает изображения, видеообзоры. Редактирование товаров построено по принципу википедии. За наполнение базы начисляется рейтинг.

Мое участие:

Проект полностью разработан мной. Кроме самого сайта было решено много задач по парсингу и автоматической обработке большого количества товаров. Создан REST API для загрузки товаров. Создано 5 отдельных вспомогательных REST-сервисов.
